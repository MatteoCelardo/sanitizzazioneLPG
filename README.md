# Sanitizzatore LPG Neo4J

## Descrizione

Questo progetto si pone lo scopo di creare un software che, fornite informazioni su cosa sia sensibile all'interno di un LPG Neo4J, permetta di sanitizzare la base di dati andando a creare autonomamente le query da inviare al DBMS, velocizzando le operazioni richieste ad un amministratore di rete che avesse la necessit√† di condividere una base di dati anonimizzata.
L'applicativo, sviluppato interamente in C\#, richiede all'amministratore di creare un file JSON contenete le specifiche che permetteranno di creare le query di sanitizzazione tramite la definizioni di quali nodi e/o relazioni rappresentino in toto o in parte un dato sensibile.
